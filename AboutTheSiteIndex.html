About the Site

This website is an active website that has records of airplane crashes that have happened around the globe. It will have the database from since commercial aircrafts have been in use. This website will only cover crashes from commercial planes and not any other planes. This website will be updates on the basis of requests. This website will detail information from crashes such as number of casualties, survivors, and details about the plane. It will also include a summary of how the crash occurred and if any investigation was done and if there was an investigation then what information has been gathered.That will the conclusion.
